Design Principles
Dark Theme
Deep void backgrounds — no pure black
Translucent color fills on blocks (20–30% opacity)
Radial ambient glows on key elements
Cherry petal floaters for atmosphere
Text hierarchy: moonveil → lavender → amethyst → umbra
Teal = actions/nav; cherry = primary/WOD; purple = strength
Light Theme
Lavender parchment bg — not pure white
Pastel fills on block chips; full-white card surfaces
Soft elevation shadows (box-shadow) instead of glows
Same petal floaters at higher opacity
Text: midnight → elderberry → iris → mist
Cherry/teal darken on light for WCAG contrast
Shared Rules
Display font: Amarante (banner, section headers, calendar nums)
Body font: Raleway 400/700/800/900 (all UI text)
Border-radius: 6–12px on cards; 4–6px on chips
Accent borders: 28–40% opacity versions of accent color
Auto-fill RM badge → teal; manual RM badge → gold
No pure #000 or #fff anywhere in UI
Block Family Colors
WOD / MetCon / For Time / AMRAP → Cherry Pink
EMOM → Spirit Teal
Força / LPO / Core / Acessórios → Twilight Purple
Aquecimento / Skill / Cardio / Mobilidade → Leaf Green
Benchmark → Blossom Gold
